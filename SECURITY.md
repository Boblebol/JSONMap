# Security Policy

## Supported Versions

Only the latest version of JSONMap is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please send an e-mail to **alexandre.enouf@gmail.com**. All security vulnerabilities will be promptly addressed.

Please include:
- A description of the issue.
- Steps to reproduce.
- Potential impact.

We appreciate your help in keeping JSONMap secure!
